FAQ

How to use this app?

Open this app from intent (open browser link) which includes supported URL. This app will show preview of picture, gif video or mp4 video from the URL. You can do some actions (e.g. download, open in other browser ...) on previewed media.

Cannot choose PhotoLinkViewer from Twitter links

If Twitter official app is installed, it always handles Twitter's URL since "App links Verification", Android 6.0's new feature. In order to handle Twitter URL with PhotoLinkViewer, you have to change Twitter's App links setting.

Change Twitter's App links setting

It may not be the same way on some devices.

  1. Launch Settings app and choose "Apps"
  2. Select "Twitter" from list of apps
  3. Select "Open by default"
  4. Change "Open supported links" to "Ask every time" from "Open in this app"
